# 🚀 Object Detector using YOLOv8

A real-time object detection application built using **Python**, **OpenCV**, and **Ultralytics YOLOv8**.

## 📌 Features
- Real-time object detection
- Detects multiple object classes
- Displays bounding boxes and confidence scores
- Lightweight YOLOv8 Nano model
- Easy to run and customize
---

## 🛠️ Tech Stack
- Python
- OpenCV
- Ultralytics YOLOv8
---

## 📂 Project Structure
```
object_detector/
│── object_detector.py
│── yolov8n.pt
│── requirements.txt
│── README.md
```

## ⚙️ Installation
Clone the repository
```bash
git clone https://github.com/nandinijha9939-dot/object_detector.git
```
Go into the folder
```bash
cd object_detector
```
Install dependencies
```bash
pip install -r requirements.txt
```
Run
```bash
python object_detector.py
```
