# Object Detector using YOLOv8

A real-time object detection application built using Python, OpenCV, and the YOLOv8 model from Ultralytics.

## Features
- Real-time object detection
- Uses YOLOv8 Nano (`yolov8n.pt`)
- Displays bounding boxes and confidence scores
- Fast and lightweight

## Tech Stack
- Python
- OpenCV
- Ultralytics YOLOv8

## Installation

Clone the repository:

```bash
git clone https://github.com/nandinijha9939-dot/object_detector.git
cd object_detector
```

Install dependencies:

```bash
pip install ultralytics opencv-python
```

Run:

```bash
python object_detector.py
```
